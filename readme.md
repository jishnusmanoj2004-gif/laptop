# 🌐 Landing Page Project

## 📌 Overview

This project is a simple, modern landing page built using **HTML** and **Tailwind CSS**. It features a clean navigation bar, a full-screen background image, and a bold hero section designed to promote a token-based platform or DAO concept.

---

## 🚀 Features

* Responsive navigation bar
* Custom fonts using Google Fonts (Orbitron & Poppins)
* Tailwind CSS styling
* Full-screen background image
* Hero section with call-to-action buttons
* Font Awesome icons integration

## ⚙️ Setup Instructions

1. Download or clone the repository.
2. Ensure all files (especially `sky.jpeg`) are in the correct directory.
3. Open `index.html` in your browser.

---

## 🎨 Customization

* **Background Image:**
  Replace `sky.jpeg` with your preferred image.

* **Text Content:**
  Modify headings and buttons inside the `<nav>` sections.

* **Colors & Styling:**
  Update Tailwind classes directly in the HTML.
